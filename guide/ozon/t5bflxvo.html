Ozon ID

Ozon ID是一种通过手机号码进行身份验证的服务，无需输入密码即可登录Ozon。


确保只有您可以访问 Ozon ID:

  • 请勿绑定您的员工或家庭成员使用的电话。
  • 不要将您的 Ozon ID 连接到其他人的帐户，哪怕他们是您商店的员工。


您用来登录的电话号码不会对任何人可见。我们不会向该号码发送广告信息——它仅用于登录授权。

在注册过程中，我们将创建一个新的Ozon ID，或者如果您已经有与手机号码绑定的买家或卖家账户，我们将使用现有的账户。 如果您有多个账户，您可以将它们绑定到同一个Ozon ID上，以便在不输入密码的情况下在它们之间切换。


通过Ozon ID登录个人中心

要在新的浏览器或新设备上无需密码登录，请在首次访问时输入您的登录名和密码。 之后，您就可以通过Ozon ID进行登录。

在登录界面上，输入与Ozon ID绑定的手机号码。 我们将向该手机号码发送验证码或拨打电话。 您无需接听电话，只需输入手机号码的最后6位数字。

如果您已经通过Ozon ID登录过，就无需密码了。系统将记住您，因此无需再次进行身份验证。


将另一个手机号码绑定到Ozon ID 

1.在右上角点击  并选择 设置。


2.进入 帐户 部分。


3.点击 转到 Ozon ID 用户信息。


4.您将进入 Ozon ID 管理页面。点击电话号码旁边的 更换 按钮并输入我们发送到您电话号码上的验证码。


5.输入新的电话号码并通过验证码对其进行确认。


将多个账户绑定到Ozon ID

1.点击屏幕右上方的商店名称。

 


2.在 添加公司 窗口中，指出现有公司。

   


3.指定要添加商店的电子邮件地址和您想添加的商店的密码。 单击登录。

  

您将看到绑定至您的 Ozon ID 的所有公司。点击 下一步 来进入个人中心。


解除Ozon ID与个人中心的绑定 

所有帐户现在都绑定到 Ozon ID，无法断开。 如果您将错误的Ozon ID绑定到账户上并且没有管理员权限，请要求经理或管理员向另一封电子邮件发送邀请，并删除旧帐户。

如果是您自己错误地绑定了Ozon ID，请向自己发送邀请至另一个地址，并将正确的Ozon ID绑定到新地址上。 然后将主管角色转移至新账户，并删除旧账户。


可能遇到的问题 

1.未收到包含与Ozon ID 绑定链接的邮件

如果您无法用此号码登录，并且电子邮件尚未到达，请检查“垃圾邮件” 箱和“促销活动邮件” 箱。

如果您在设置 Ozon ID 时遇到困难，请通过 global.registration@ozon.ru 联系我们。


2.无法登录个人中心

  • 请刷新页面并尝试再次进入。

  • 请删除cookie文件并清空浏览器存储器：

  • 请通过其他浏览器进入。


3.未收到验证码

  • 请确认您的手机已开机并且在网络范围内。
  • 请确认您的手机号码正确。

有时不会立即收到短信。若在20分钟内仍未收到短息，请再次尝试获取验证码。


4.如果手机号码不可用，如何登录账户

如果您是经理

请发邮件至marketplace.crossborder@ozon.ru，我们将帮助您恢复。


如果您是其他角色

  • 请要求商店经理将无法登录的账户从员工名单中移除。如果您可以访问多个公司，则必须从每个公司中删除该账户。
  • 请在Ozon ID授权 页面点击通过邮箱进入。请指定绑定至Ozon ID的邮箱，然后输入邮件中的验证码。若您授权失败，则表示该账户仍绑定至商店。请联系经理。
  • 请在电话字段点击更改。
  • 请点击无需确认更改。
  • 请输入新的电话号码并点击获取验证码。
  • 请输入验证码。

    • 在您更换新的电话号码后，请让经理给您发送新的邀请。


    5.如果电子邮箱不可用，如何登录账户

    如果您是经理

    请在Ozon ID授权页面点击恢复账户。请填写表格以恢复访问权限。


    如果您是其他角色

  • 请要求商店经理将无法登录的账户从员工名单中移除。如果您可以访问多个公司，则必须从每个公司中删除该账户。
  • 请在Ozon ID授权通过电话号码进入。若您授权失败，则表示该账户仍绑定至商店。请联系经理。
  • 请在邮箱字段点击更改。
  • 请输入新的邮箱并点击获取验证码。我们会将其发送至新的邮箱。
  • 请输入邮件中的验证码。
